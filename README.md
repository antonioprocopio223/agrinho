🌱 Agro Forte: Futuro Sustentável

Projeto desenvolvido durante a aula de Programação da professora Regiane Heil Plem.

📖 Sobre o Projeto

O Agro Forte: Futuro Sustentável é uma página web criada com HTML, CSS e JavaScript que apresenta a importância do equilíbrio entre a produção agrícola e a preservação do meio ambiente.

O objetivo do projeto é demonstrar, de forma simples e interativa, como práticas sustentáveis podem contribuir para o desenvolvimento do agronegócio sem comprometer os recursos naturais para as futuras gerações.

🎯 Objetivos
Promover a conscientização sobre a sustentabilidade no agronegócio.
Demonstrar a utilização das tecnologias HTML, CSS e JavaScript.
Criar uma interface simples, organizada e responsiva.
Apresentar uma interação básica por meio de atualização de indicadores de produção e meio ambiente.
🛠️ Tecnologias Utilizadas
HTML5 – Estrutura da página.
CSS3 – Estilização e layout.
JavaScript – Interatividade e atualização dos indicadores.

🌍 Mensagem do Projeto

Produzir mais não significa preservar menos. O futuro do agronegócio depende do equilíbrio entre produtividade, inovação e responsabilidade ambiental.
